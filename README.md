# Artificial Intelligence Course Assignments

This repository contains my completed assignments for the Artificial Intelligence course. The assignments are based on the curriculum and exercises provided in the following public GitHub repository:

**[jinqijinqi/Artificial-Intelligence-Course](https://github.com/jinqijinqi/Artificial-Intelligence-Course)**

---

## My Assignments

This section provides a brief description of the files included in this repository, organized by topic or week.

* **`astar_constrained_transport.py`**: An A\* search algorithm implementation for solving a constrained transportation problem.
* **`dice-mdp.py`**: A solution for a Markov Decision Process (MDP) problem involving a dice game.
* **`mdp_gridworld_implementation.py`**: Implementation of an MDP for a simple gridworld environment.
* **`linear.py` & `regression_nonlinear.csv`**: Solutions and data for linear regression assignments.
* **`tang_poetry_gpt.py`**: A program using a language model to generate Tang-style poetry.
* **`week1_solution.py`**: My solution for the assignments from Week 1.
* **`week2_solution.py`**: My solution for the assignments from Week 2.
* **`week2_2_solution.py`**: My solution for another set of assignments from Week 2.
* **`training_history.png`**: An image showing the training progress of a model.
* **`venv/`**: A virtual environment for the project dependencies.
